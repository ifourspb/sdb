Работает с плагином Contact Form 7, отправляет СМС через API ByteHand, когда форма отправлена.

== Описание ==
В настройках формы плагина Contact Form 7 появляется форма настройки SMS.

Вам нужен аккаунт на bytehand.com и средства на счету для работы.

= Требования =

* Wordpress 3 или выше

* [Contact Form 7](http://wordpress.org/extend/plugins/contact-form-7/) 3.4 или выше

== Установка ==

1. Скачайте файл contact-form-7-sms-bytehand-addon.zip
2. Распакуйте содержимое загруженного архива в /wp-content/plugins/
3. Активируйте плагин в меню Plugins
4. Укажите данные вашего аккаунта ByteHand в админ панели 'ByteHand SMS' > 'bytehand настройки'
5. Настройки параметры СМС-уведомлений для каждой формы плагина Contact Form 7
